git 基础命令行操作

  • git init //初始化本地git仓库
  • git add //添加
  • git status //查看状态
  • git commit //提交
  • git push //推送到仓库
  • git pull //从远程仓库拉取数据
  • git clone //从远程仓库拷贝数据
  • git branch //创建分支 名为
  • git checkout //切换到分支
  • git checkout master //切换回主线
  • git merge //在切换回主线时 合并分支

git 安装 操作

  • https://git-scm.com //下载git 并安装
  • git --version //检查git是否安装
  • git init //初始化
  • git config --global user.name 'linlyc' //配置用户名
  • git config --global user.email 'chanlinly@qq.com' //配置邮箱
  • git add //添加文件
  • git status //查看状态
  • git rm --cached //删除文件
  • git commit //提交到本地仓库
  • 头部加备注 esc :wq //退出
  • git commit -m '备注信息' //备注提交到本地仓库
  • 创建.gitignore文件 把需要忽略的文件或文件夹名次写在.gitignore文件上 例如：log.txt ／log //忽略log.txt文件,log文件夹
  • git branch 'branch_name' //创建分支
  • git checkout 'branch_name' //切换到分支
  • git checkout master //切换到主线
  • git merge 'branch_name' //在主线时 把分支合并到主线上
  • vi 冲突文件 //merge冲突，解决冲突（vi）后，重新add commit
  • git checkout －b 'branch_name' //创建分支并切换到分支
  • git commit －am '备注信息' //add添加并提交备注
  • git reset --hard HEAD^ //返回上一版本
  • git reset --hard HEAD～n //返回前n个版本
  • git reflog //查看提交记录

github 操作

  • create a new repository //创建仓库
  • git remote //查看是否与远程连接
  • git remote remove origin //关闭与远程连接
  • git remote add origin github地址 //与远程连接
  • git pull --rebase origin master //更新远程仓库最新数据
  • git push -u origin master //提交到远程仓库
